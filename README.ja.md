# LubeLogger日本語翻訳(ja_JP)

LubeLogger向けの日本語翻訳(ja_JP)です。日本の自動車・オートバイユーザー向けにローカライズもしています。

---


## 概要説明

このリポジトリではLubeLogger用の日本語翻訳ファイル (`ja_JP.json`) を提供しています。

この翻訳は**単純に英単語をそのまま翻訳したものではありません**
日本の自動車・オートバイの個人所有者向けのローカライズに重点を置いて翻訳をしています。

この翻訳の主な目的は、元の英語テキストへの厳格な忠実さではなく、日本のユーザーにとっての使いやすさや文化的な整合性を意識し作りました。

---

## 翻訳方針

この翻訳は下記の方針で行っています :

- 日本の自動車・オートバイユーザーが一般的に使用する用語を優先
- 直訳よりも実用的な使いやすさを重視
- 日本では一般的ではない、また不明確な概念の部分は無理な翻訳をしない
- アプリケーションの動作を損なわない範囲で機能的な一貫性を維持
- 誤解を招く翻訳を導入するより、原文(en_US)のまま残すことを優先

このため、一部のラベルは英語表記から大きく異なる場合がありますが、アプリケーションの一貫性を保ち、動作を損なうことはありません。

---

## 意図的に未翻訳・部分翻訳のセクション

以下のセクションは意図的に未翻訳、または部分翻訳のままとなっています :

- 日本ではほとんど使用されないと思われるタブ(EquipmentやInspections)
- サーバー側の設定および高度なシステム設定部分
- 日本で広く認知されていない概念に基づく機能(Vehicle Mapなど)

これは質の低い翻訳や誤訳を招く恐れがあるので意図的に原文残しとしている

## インストール方法

1. このリポジトリから `ja_JP.json` をダウンロード

2.  LubeLoggerを開き、以下へ移動
   - Settings
   - Language
   - Manage Languages

3. **UpLoad**をクリックし、ダウンロードし保存した `ja_JP.json` を選択

4. 必要に応じて言語設定を適用

5. 言語設定で「日本語(日本)」を選択

### 注意事項

- **Language**経由でのアップロードが推奨される方法です
- ファイル名が内部ファイルとコンフリクトする場合があります。
  その場合は `ja_JP_custom.json` などにリネームしてからアップロードしてください。
- 内部ディレクトリへの手動でのファイル配置は不要です
